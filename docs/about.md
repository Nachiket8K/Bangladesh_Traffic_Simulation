# About

## About this project

This repository documents a course project on **Bangladesh transport network simulation, bridge disruption, and network analysis**. The work develops across multiple assignments that progressively move from data cleaning and preprocessing to network construction, simulation, and interpretation of experiment results.

The overall aim of the project is to understand how road and bridge infrastructure affects freight movement, and how bridge condition and disruption can influence travel times, bottlenecks, criticality, and vulnerability in the transport network.

## Course context

The repository was developed as part of **EPA1352 Advanced Simulation**. Across the assignments, the group worked through four connected stages:

1. **Data Quality / Data Cleaning** — understanding and repairing road and bridge datasets
2. **Component Building** — creating the first corridor-scale simulation model
3. **Network Model Generation** — expanding the corridor model into a graph-based road network
4. **Network Analysis** — analyzing traffic, bridge importance, criticality, and vulnerability

This documentation site was created to make that project understandable to readers who were not part of the original coursework.

---

## What this site documents

The documentation site summarizes the project as an end-to-end workflow:
- how the infrastructure data is cleaned and prepared,
- how roads and bridges become a network model,
- how truck movement and bridge delays are simulated,
- how experiments are designed and interpreted,
- how the repository can be understood and reproduced.

It is intended both as:
- a readable explanation of the project,
- and a guide for navigating the code, notebooks, data, and outputs in the repository.
