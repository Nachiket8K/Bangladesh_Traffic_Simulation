# Method & Model

This section explains how the project is built, starting with the **Assignment 1 data-cleaning and preprocessing work** that prepares the road and bridge datasets used by later network and simulation stages.

It covers:
- Data cleaning and preparation
- Graph/network construction
- Simulation logic and event handling (bridge breakage → delay)
