# Bangladesh Transport Network Simulation

A project about **road freight movement, bridge disruption, and network performance in Bangladesh**. The repository combines infrastructure data preparation, network construction, simulation, scenario analysis, and visualization to study how bridge condition affects truck movement and system performance.

## What this site contains

- **Project Overview** — the problem, scope, and full workflow of the repository
- **Method & Model** — data preprocessing, network construction, and simulation logic
- **Experiments & Results** — scenario design, metrics, findings, and interpretation
- **Visualization** — an interactive or communicative view of the network and scenarios
- **Reproducibility** — where the main scripts, datasets, and outputs come from

## Suggested reading order

1. **Project Overview** to understand the system boundary and pipeline.
2. **Method & Model** to see how the project is built.
3. **Experiments & Results** to understand what was tested and what was found.
4. **Visualization** for a faster intuitive understanding of the network and disruptions.
5. **Reproducibility** if you want to trace the workflow back to the source notebooks and scripts.

> Tip: If you want the quickest conceptual understanding, start with **Project Overview** and then open the **Visualization**.
